# Финансовый аналитический проект

## Описание
Учебный проект для работы с финансовыми данными. Демонстрирует обработку текста, фильтрацию данных и работу с файлами форматов TXT, CSV и JSON.

## Функционал
- Очистка и анализ текста (нормализация, подсчёт слов)
- Фильтрация списка пользователей по имени и городу
- Создание и чтение текстовых файлов
- Работа с CSV-файлами (финансовые показатели)
- Работа с JSON-файлами (конфигурация проекта)

## Финансовые данные (CSV)
Файл `finance.csv` содержит 6 колонок:
- статья - наименование показателя
- дебет - дебетовый оборот
- кредит - кредитовый оборот
- сальдо - конечное сальдо
- период - отчётный период
- валюта - валюта учёта (RUB)

1. Клонировать репозиторий
2. Создать виртуальное окружение:
```bash
python -m venv .venv
Активировать окружение:

Windows: .venv\Scripts\activate

Mac/Linux: source .venv/bin/activate

Запустить проект:

bash
python main.py
